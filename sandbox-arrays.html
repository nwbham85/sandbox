map()

What it does:

Transforms each item in the array.

returns:

A new array. Does not modify original.

Transform Objects within arrays:

              
                const users = [
                  {name: "Alice", age: 30},
                  {name: "Bob", age: 40},
                  {name: "Charlie", age: 25}
                ];

                const names = users.map(user => user.name);

                console.log(names);

Creating new objects

              
                const users = [
                  {name: "Alice", age: 30},
                  {name: "Bob", age: 40}
                ];

                const profiles = users.map(user => ({
                  username: user.name,
                  isAdult: user.age >= 18
                }));

                console.log(profiles);
